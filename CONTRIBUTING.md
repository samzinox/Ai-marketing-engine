# Contributing to AI Marketing Engine

We welcome contributors who care about clarity over hype.

## Ways to Contribute

- Improve existing AI prompts
- Add new marketing workflows
- Share real product examples
- Fix typos or documentation

## Contribution Rules

- Keep everything simple
- No buzzwords or guru language
- Practical > theoretical
- Developers first

## How to Contribute

1. Fork the repo
2. Create a branch
3. Make your changes
4. Submit a PR with a clear explanation

Small improvements are welcome.
